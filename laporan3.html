Laporan Hasil 3

Area Permasalahan dan Rancangan

Area permasalahan dari projek ini adalah bagaimana cara memberikan interface yang mudah bagi user (pembeli di lab bisnis AN) untuk memesan makanan/minuman sesuai dengan keingingannya. Sesuai dengan permasalahan tersebut user menginginkan adanya sistem pemesanan yang mempermudah user dalam memilih makanan/minuman yang diinginkan.

Melihat area permasalahan tersebut, maka kami membuat sebuah rancangan aplikasi berbasis website yang memfasilitasi user dalam memilih makanan/minuman yang diinginkan. Untuk memudahkan user dalam mencari produk favorit, kami merancang tampilan rating dalam setiap menu makanan/minuman yang disediakan.

Usability Goals

  • Effectiveness (how good a product is at doing what it is supposed to do)
    • Apakah user dapat menemukan makanan/minuman yang diinginkan?
    • Apakah user dapat melakukan pemesanan produk sampai akhir task?
  • Efficiency (the way a system supports users in carrying out their tasks)
    • Apakah seluruh task dapat dilakukan dengan cepat?
    • Apakah ada eksekusi task yang justru akan lebih cepat dieksekusi secara manual (tanpa bantuan aplikasi)?
  • Safety (protecting the users from dangerous conditions and undesirable situations)
    • Apakah user melakukan banyak kesalahan ketika berinteraksi dengan sistem?
  • Utility (extent to which the system provides the right kind of functionality so that users can do what they need or want to do)
    • Apakah aplikasi sudah memberikan fitur yang dibutuhkan?
  • Learnability (how easy a system is to learn to use)
    • Bagaimana performa user ketika pertama kali mengerjakan task pada aplikasi?

Teknik Evaluasi

Pada evaluasi kali ini kami menggunakan tiga teknik, yaitu:

  • Observasi dan monitoring
  • Opini user
  • Key Stroke Level Model

Keterlibatan User

Pada evaluasi kali ini, user terlibat dalam:

  • Memberikan kritik dan juga saran terhadap prototipe sistem yang dibuat
  • Mencoba prototipe sistem yang dibuat

Task yang harus dikerjakan user

Ada beberapa task yang harus dikerjakan user:

  • Memilih menu makanan/minuman
  • Melihat detail makanan/minuman
  • Memilih metode pembayaran
  • tambahkan kalau kurang
